# Pragadeesh Elangaivendhan — Portfolio

A professional portfolio website built with React, showcasing skills, experience, and projects.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Customize

Edit `src/data/portfolio.js` to update your personal info, skills, experience, education, and projects.

## Tech Stack

- React 19
- Vite
- Plain CSS (no external UI libraries)
